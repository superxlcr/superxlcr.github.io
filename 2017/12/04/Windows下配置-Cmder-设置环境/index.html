Cmder 是一款好用的 Console Emulator，其官网为：
http://cmder.net/

下载完后，我们可以在 Window 下配置我们的 Cmder 了

win + R 启动 Cmder

我们可以在 Window 环境变量的 PATH 中添加我们Cmder的路径，以后就可以通过 win + R 输入相关名称来启动我们的Cmder了
